== Description ==

Page Assets Optimizer helps you improve your website performance by:

* Disabling individual assets or full plugins on specific pages
* Automatically optimizing images (convert JPG/PNG to WebP/AVIF)
* Minifying CSS and JavaScript files
* Tracking performance improvements

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/page-assets-optimizer` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to 'Page Assets Optimizer' in the admin menu to configure

== Frequently Asked Questions ==

= How do I disable assets on specific pages? =

Go to Page Assets Optimizer > Optimizer in your WordPress admin. Select the page/post and choose which assets to disable.

= Does this plugin minify CSS/JS? =

Yes, you can enable minification in the Settings tab.
