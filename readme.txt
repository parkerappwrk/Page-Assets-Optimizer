== Description ==

Page Assets Optimizer provides tools to manage CSS and JavaScript assets on a per-page basis.

It allows administrators to:

* Disable individual assets or entire plugins on specific pages
* Configure image optimization settings, including conversion to modern formats such as WebP or AVIF
* Minify CSS and JavaScript files
* View asset usage and optimization settings within the admin panel

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/page-assets-optimizer` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to 'Page Assets Optimizer' in the admin menu to configure

== Frequently Asked Questions ==

= How do I disable assets on specific pages? =

Go to Page Assets Optimizer > Optimizer in your WordPress admin. Select the page/post and choose which assets to disable.

= Does this plugin minify CSS/JS? =

Yes, you can enable minification in the Settings tab.
